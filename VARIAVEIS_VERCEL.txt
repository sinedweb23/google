===========================================
INSTRUÇÕES:
===========================================

1. Acesse: https://vercel.com/dashboard
2. Selecione seu projeto (ou crie novo)
3. Vá em: Settings > Environment Variables
4. Clique em: Add New
5. Cole cada variável acima
6. Marque todos os ambientes (Production, Preview, Development)
7. Clique em Save

===========================================
ONDE OBTER OS VALORES:
===========================================

SUPABASE_SERVICE_ROLE_KEY:
→ https://app.supabase.com/project/hvrrixmsijkehdlaxdsi/settings/api
→ Role: service_role
→ Copie a chave (começa com eyJ...)

GOOGLE_SERVICE_ACCOUNT_EMAIL e GOOGLE_PRIVATE_KEY:
→ Google Cloud Console
→ IAM & Admin > Service Accounts
→ Selecione seu Service Account
→ Keys > Add Key > Create new key > JSON
→ Do JSON, copie:
   - client_email → GOOGLE_SERVICE_ACCOUNT_EMAIL
   - private_key → GOOGLE_PRIVATE_KEY (com \n)

API_KEY_INTERNA:
→ Gere uma chave forte:
   Windows PowerShell: [Convert]::ToBase64String((1..32 | ForEach-Object { Get-Random -Minimum 0 -Maximum 256 }))
   Ou use: https://www.random.org/strings/
